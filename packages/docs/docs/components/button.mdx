# Button

Buttons allow users to take actions, and make choices, with a single tap.

```bash
import { Button } from '@sui/core'
```

## Basic button

The `Button` comes with three variants: default, text, and outlined.

<App>

<Example>
  <Button>Default</Button> <Button text>Text</Button> <Button outlined>Outlined</Button>
</Example>

</App>

```tsx
<Button>Default</Button>
<Button text>Text</Button>
<Button outlined>Outlined</Button>
```

### Default button

Default buttons are high-emphasis, distinguished by their use of elevation and fill. They contain actions that are primary to your app.

<Example>
  <Button>Default</Button>
  <Button disabled>Default</Button>
</Example>

```tsx
<Button >Default</Button>
<Button disabled>Default</Button>
```

## Button Props

| Attribute | Type                | Description    | Default |
| --------- | ------------------- | -------------- | ------- |
| block     | boolean             | 是否全屏宽度   | false   |
| disabled  | boolean             | 是否禁用       | false   |
| text      | boolean             | 文本按钮       | false   |
| outlined  | boolean             | 边框按钮       | false   |
| icon      | boolean             | 图标按钮       | false   |
| rounded   | boolean             | 圆边按钮       | false   |
| radius    | string              | 圆边度数       |         |
| padding   | string              | 内边距         |         |
| color     | string              | 自适应按钮颜色 |         |
| css       | `Themed<CSSObject>` | css 样式       |         |
